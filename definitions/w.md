# W

### While Loop
A very specific type of flow control which allows a programmer to control the number of times a particular line of code is executed. The example below adds all of the numbers from 0 to 10.
```
Integer sum = 0;
Integer i = 0;
while(i <= 10) {
  sum = sum + i;
}
```
